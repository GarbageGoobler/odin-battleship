## Design Notes

This color scheme follows the [Catppuccin Style Guide](https://github.com/catppuccin/catppuccin/blob/main/docs/style-guide.md) for Mocha flavor:

- **Base** - Darkest background color
- **Mantle** - Secondary background for elevated elements
- **Surface0** - Surface elements (buttons, input fields)
- **Text** - Primary text color for readability
- **Blue** - Primary accent color for main operations
- **Red** - Accent color for destructive actions (clear, delete)
- **Peach** - Action color for the equals button
- **Lavender** - Border accent for the screen
- **Mauve** - Glow effect for the calculator body

All accent colors use **Base** for text to ensure proper contrast as per Catppuccin guidelines.
