# JS Webpack Starter Template

Minimal Webpack + ESLint starter for vanilla JavaScript projects.

## Use this template for a new project

1. Click **Use this template** on GitHub and create your new repo.
2. Clone the new repo and install dependencies:

```bash
git clone <your-new-repo-url>
cd <your-new-repo-name>
npm install
```

3. Start development server:

```bash
npm run dev
```

4. Build production bundle:

```bash
npm run build
```

5. Start coding in `src/index.js` and `src/template.html`.
